#  REMIND — A Personal Feedback Loop to Prevent Repeated Mistakes

REMIND is a personal decision-learning tool that helps users avoid repeating past mistakes
by turning moments of regret into actionable signals. Users log situations they wish had
gone differently, and REMIND learns patterns from these experiences to provide timely,
non-judgmental reminders before similar situations occur again.

> Built as a semester project for the Responsible Computing Club at SJSU (Spring 2026).

---

##  How It Works

1. **Log** — User records a moment of regret (what happened, trigger, what they wish they did)
2. **Detect** — System finds patterns across past entries using AI similarity matching
3. **Remind** — When a similar situation is detected, user gets a gentle, non-judgmental nudge
